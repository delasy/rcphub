# rcphub-web-backend
Back-End of RcpHub

## Installation
```bash
$ cd /path/to/rcphub-web-backend

# Install Node.js dependencies
$ npm run install
```

## Available Scripts
In the project directory, you can run:

### `npm run dev`
Runs the app in the development mode.

### `npm start`
Runs the app in the production mode.

### `npm test`
Runs the test runner.
